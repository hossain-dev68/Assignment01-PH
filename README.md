# DevConf Landing Page

## Project Overview
This is a responsive landing page for a fictional developer conference called DevConf. The website is built using only HTML and CSS.

## Features
- Responsive navigation bar
- Hero banner section
- Speakers section
- Pricing (Secure Your Spot) section
- Before & After comparison section
- Footer

## Technologies Used
- HTML5
- CSS3
- Google Fonts
